# 위스키 테이스팅 노트 그래프 설계

## 개요

레이더 차트로 위스키 풍미 프로파일을 시각화하는 기능. 유저가 슬라이더로 각 축의 점수를 조절하면 차트가 실시간 반영되고, 완성된 노트를 이미지로 저장할 수 있다.

## 핵심 스펙

| 항목             | 결정                                           |
| ---------------- | ---------------------------------------------- |
| 차트 타입        | 레이더 차트 (스파이더 차트)                    |
| 축 구성          | 고정 5개: 바디, 피티, 스위트, 프루티, 스파이시 |
| 점수 범위        | 0~10점                                         |
| 입력 방식        | 슬라이더 연동 (실시간 차트 반영)               |
| 저장 방식        | 이미지 다운로드 (PNG)                          |
| 이미지 포함 정보 | 차트 + 위스키 이름 + 날짜 + 각 점수 + 메모     |
| 진입 경로        | 위스키 상세 페이지 → 테이스팅 노트 페이지      |

## 페이지 구조

**경로:** `/tasting-note/[alcoholId]`

```
┌─────────────────────────────┐
│  위스키 정보 (이름, 이미지)    │
├─────────────────────────────┤
│                             │
│     레이더 차트 (중앙)        │
│                             │
├─────────────────────────────┤
│  슬라이더 5개 (각 축별)       │
│  - 바디: ───●─── 7          │
│  - 피티: ─●───── 3          │
│  - 스위트: ────●── 8        │
│  - 프루티: ──●──── 5        │
│  - 스파이시: ─●───── 4      │
├─────────────────────────────┤
│  메모 입력 (텍스트 영역)      │
├─────────────────────────────┤
│  [이미지로 저장] 버튼         │
└─────────────────────────────┘
```

## 컴포넌트 구조

```
src/app/(primary)/tasting-note/
├── [alcoholId]/
│   └── page.tsx                    # 테이스팅 노트 페이지
│   └── _components/
│       ├── TastingNoteForm.tsx     # 전체 폼 컨테이너
│       ├── TastingRadarChart.tsx   # 레이더 차트
│       ├── TastingSliderGroup.tsx  # 슬라이더 그룹
│       └── TastingNoteCapture.tsx  # 이미지 저장용 캡처 영역
```

## 기술 스택

- **차트:** Recharts (RadarChart 컴포넌트)
- **이미지 캡처:** html2canvas
- **상태 관리:** useState (로컬 상태)

## 데이터 흐름

```
위스키 상세 페이지
    ↓ (alcoholId를 URL 파라미터로 전달)
테이스팅 노트 페이지
    ↓ (alcoholId로 위스키 정보 fetch)
폼 작성 (슬라이더 조작 → 차트 실시간 업데이트)
    ↓
이미지 저장 버튼 클릭
    ↓
html2canvas로 캡처 → PNG 다운로드
```

## 데이터 타입

```typescript
interface TastingNote {
  body: number; // 0-10
  peaty: number; // 0-10
  sweet: number; // 0-10
  fruity: number; // 0-10
  spicy: number; // 0-10
  memo: string;
}

interface TastingNotePageProps {
  alcoholId: number;
  alcoholName: string;
  alcoholImage: string;
}
```

## 구현 범위 (MVP)

1. 테이스팅 노트 페이지 생성
2. 레이더 차트 컴포넌트 (Recharts)
3. 슬라이더 5개 연동
4. 메모 입력 텍스트 영역
5. 이미지 저장 기능 (html2canvas)
6. 위스키 상세 페이지에 진입 버튼 추가

## 설치 패키지

```bash
pnpm add recharts html2canvas
```
